## Common setup

Clone the repo and install the dependencies. 

```bash
    git clone https://github.com/TyrkoDev/BlagueDeMerde.git
    cd BlagueDeMerde
```
Install front dependencies
```bash
    cd front
    npm install
```
Install back dependencies
```bash
    cd server
    npm install
```
- Jenkins build : OK
